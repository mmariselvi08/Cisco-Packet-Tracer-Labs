# Cisco Packet Tracer - Basic LAN Network

## Objective
Create a basic LAN using Cisco Packet Tracer.

## Devices Used
- 2 PCs
- 1 Cisco 2960 Switch

## IP Configuration
- PC0: 192.168.1.1
- PC1: 192.168.1.2
- Subnet Mask: 255.255.255.0

## Verification
- Successfully tested connectivity using the ping command.

## Software
- Cisco Packet Tracer
